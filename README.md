# solanaj-programs

SolanaJ implementations of popular Solana programs, including the Project Serum DEX.

# Installation
To install all modules:
```xml
<dependency>
        <groupId>ch.openserum</groupId>
        <artifactId>solanaj-programs</artifactId>
        <version>1.0-SNAPSHOT</version>
</dependency>
```
