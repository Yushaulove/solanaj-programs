# solanaj-programs

SolanaJ implementations of popular Solana programs, including the Project Serum DEX.

# Installation
1. Add Maven dependency:

```xml
<dependency>
        <groupId>com.mmorrell</groupId>
        <artifactId>solanaj-programs</artifactId>
        <version>1.8</version>
</dependency>
```
